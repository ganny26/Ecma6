# JS Console

  - Simple native JS console editor 

### Live Demo

App installed in heroku

https://selvaconsole.herokuapp.com/


# How to run 

```
$ yarn install
$ yarn build
$ yarn start

```

### Tech
* Native JavaScript
* ESLint - to identify lint errors
* Webpack - to bundle module code 
* Babel - for transpiling 
* Pretty JS - for syntax highlighting
